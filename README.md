# Playwright Learning Project

##  Überblick
Dieses Projekt dient als Lern- und Übungsumgebung für Playwright mit Python.  
Es enthält erste Tests, Page-Object-Strukturen, Utilities sowie eine vorbereitete Pytest-Konfiguration.

##  Technologien
- Python
- Playwright
- Pytest
- Pytest Fixtures
- GitHub Actions (optional erweiterbar)
- Requirements über `requirements.txt`

## Projektstruktur

Playwright_Learning/
│
├── Pages/              # Page Object Dateien
├── Tests/              # Testfälle
├── Utilities/          # Hilfsfunktionen, Helper, Tools
│
├── conftest.py         # Fixtures für Browser, Kontext, Setup
├── pytest.ini          # Pytest-Konfiguration
├── requirements.txt    # Python-Abhängigkeiten
└── .github/workflows/  # (optional) CI/CD Workflows
